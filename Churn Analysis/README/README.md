📊 Churn Behavior Analysis Dashboard
🚀 Project Overview

This project presents an interactive data visualization dashboard built to analyze customer churn behavior. The dashboard helps identify key factors influencing churn and provides actionable insights to improve customer retention and revenue growth.

🎯 Objective
Analyze customer churn patterns
Identify high-risk customers
Understand revenue impact due to churn
Support data-driven business decisions
🛠️ Tools & Technologies Used
Power BI – Dashboard creation & visualization
Data Analysis – Data cleaning & transformation
DAX – Calculated measures & KPIs
Excel/CSV Dataset – Data source
📌 Key Features
📍 Total Customers & Churned Customers Overview
📍 Churn Analysis by Contract Type
📍 Customer Distribution by Payment Method
📍 Revenue Analysis (Active vs Loss)
📍 Churn Rate by Gender
📍 Customer Churn by Tenure
📍 Tech Support Impact on Churn
📊 Key Insights
Customers with month-to-month contracts show higher churn rates
Lack of tech support significantly increases churn probability
Electronic check payment method users churn more frequently
Majority of revenue loss is contributed by short-term customers
